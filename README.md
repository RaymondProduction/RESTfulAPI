# Simple server when use REST API

## Step 1, install modules
```
$ npm install
```

## Step 2, start server
```
$ node src/js/app.js
```

This server automatically find ***index.html*** because scheme of directory such:
```
project
└── src
    ├── index.html
    └── js
        ├── app.js
        └── routes
            └── routes.js

```
